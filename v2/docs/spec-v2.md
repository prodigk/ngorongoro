# ngorongoro v2 specification

## 목표

브랜드 소개 중심인 현재 사이트를 확장해 작품 탐색과 외부 구매 연결이 가능한 독립 사이트를 제작합니다.

## 페이지

- Home: 대표 작품과 브랜드 첫인상
- Gallery: 작품 목록, 상태, 외부 구매 링크
- About: 브랜드와 작업 이야기
- Contact: 이메일, Instagram, 입점 및 협업 문의

## 기본 요구사항

- 반응형 웹
- 화이트 기반의 단정한 갤러리 UI
- 키보드 탐색과 명확한 포커스 상태
- 작품 이미지 대체 텍스트 제공
- 외부 구매 링크에 새 창 안내 및 보안 속성 적용
- 현재 운영 사이트와 코드 및 자산 분리

## 추후 결정

- 작품 상세 페이지 도입 여부
- 구매처와 작품별 URL
- 문의 폼 또는 이메일 링크 사용 여부
- 다국어 지원 여부
- 최종 도메인 및 배포 경로
